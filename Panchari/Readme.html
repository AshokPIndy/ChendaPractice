Panchari Melam Help

Introduction

You can follow the highlighted letters in Karaoke style to practice Panchari Melam. Be sure to use a large screen!

Before playing you need to load a Melam, which is just a text file specifying what to play for each Thalavattam. like Nila, Idukol etc. You can download a sample here.

The page will NOT validate if the melam is valid. If invalid, it will not work properly.

Melam will start in 3rd Kalam and will switch to 4th after Theerukalam


Speed of 48 will play 2 thalavattams of 3rd kalam in approximately one minute.
